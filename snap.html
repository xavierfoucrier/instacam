Instacam

Instacam is a jQuery plugin to perform instant canvas video through the WebRTC API with a fresh touch of CSS filters.

Snap demo

Capture the viewport image data. Enjoy!


Source code

$(function() {
  $('#canvas1').instacam();
  
  $('input[name="snap"]').click(function() {
    var snapshot = $('#canvas1').data('instacam').snap();
    var thumbnail = document.createElement('canvas');
    thumbnail.setAttribute('width', $('#canvas1').instacam('width'));
    thumbnail.setAttribute('height', $('#canvas1').instacam('height'));
    $(thumbnail)[0].getContext('2d').putImageData(snapshot, 0, 0);
    $('.thumbs').append(thumbnail);
  });
});

Documentation

Instacam can do much more than display a camera stream! CSS filtering, custom filtering applied on HTML5 canvas, snapshots, etc. View the project on Github to see the documentation about available options, methods and callbacks.
